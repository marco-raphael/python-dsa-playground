Python DSA: Laboratory 🐍🚀
Este repositório é o meu laboratório pessoal de Estruturas de Dados e Algoritmos em Python. Aqui, documento minha evolução técnica, unindo a simplicidade do Python com o rigor da ciência da computação.

🎯 Objetivo do Projeto
Transformar teoria em código eficiente. Utilizo este espaço para dominar a lógica de organização de dados e processamento, focando sempre em escrever código limpo e testável.

📂 Organização do Laboratório
O projeto utiliza uma estrutura modular para facilitar o reuso e os testes unitários:

src/algorithms/: Implementações de lógica pura (Ordenação, Buscas, Recursão, Algoritmos Gulosos).

src/data_structures/: Implementações do zero de estruturas fundamentais (Stacks, Queues, Linked Lists, Trees).

tests/: Minha rede de segurança. Utilizo o Pytest para garantir que cada algoritmo se comporte exatamente como esperado.

[!TIP]
Em vez de métodos main poluindo as classes, cada algoritmo tem seu respectivo teste na pasta /tests. Para rodar tudo, basta um pytest no terminal! ✅

🛠️ Stack de Estudos
Linguagem: Python 3.12+

IDE: PyCharm

Framework de Testes: Pytest

Foco: Clean Code, Type Hinting (tipagem estática) e Análise de Complexidade (Big O).
