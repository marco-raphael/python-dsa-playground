# 🐍 Python DSA: Laboratory 🚀

> **"Transformando teoria em código eficiente."**

Este repositório é o meu laboratório pessoal de **Estruturas de Dados e Algoritmos (DSA)** em Python. Aqui, documento minha evolução técnica, unindo a simplicidade da sintaxe Python com o rigor analítico da Ciência da Computação.

---

## 🎯 Objetivo do Projeto
O foco principal é dominar a lógica de organização e processamento de dados, priorizando a escrita de código **limpo, tipado e testável**. É o espaço onde a complexidade de algoritmos encontra as melhores práticas de desenvolvimento.

---

## 📂 Organização do Laboratório
O projeto utiliza uma estrutura modular para facilitar o reuso e a manutenção:

* **`src/algorithms/`**: Implementações de lógica pura.
    * *Ordenação, Buscas, Recursão, Algoritmos Gulosos.*
* **`src/data_structures/`**: Implementações "from scratch" (do zero).
    * *Stacks, Queues, Linked Lists, Trees.*
* **`tests/`**: Minha rede de segurança.
    * *Suíte de testes unitários para garantir a integridade de cada implementação.*

> [!TIP]
> **Clean Code:** Em vez de métodos `main` poluindo as classes, cada algoritmo tem seu respectivo teste na pasta `/tests`. 
> Para rodar a validação completa, basta um comando:
> ```bash
> pytest
> ```
